
  # AI Sales Forecasting Dashboard

  This is a code bundle for AI Sales Forecasting Dashboard. The original project is available at https://www.figma.com/design/mB4QPLozjz4hQUaGZKczWR/AI-Sales-Forecasting-Dashboard.

  ## Running the code

  Run `npm i` to install the dependencies.
  
  
  Run `npm install lucide-react` for icons.
  

  Run `npm run dev` to start the development server.
  


  ## Backend Setup

  ## Navigate to the backend folder:

  Run `cd backend`
  
  Run the backend server:
  
  Run `python app.py`


Access the Website

To use the website, you need to sign up and log in first.
